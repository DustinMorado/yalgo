new_parser (description)
Creates a new parser object. Returns a parser with the given description, and standard help options.

Parameters:

  • description A description for the CLI program you are parsing

Returns:

    A parser table

Usage:

    parser = yalgo:new_parser('A sample program description.')

Display program help. Shows usage and details optional and positional arguments

get_arguments (arguments)
Get arguments. Takes a table similar to and by default _G.arg, and returns a table of options storing the found and default values.

Parameters:

  • arguments Sequence of arguments by default _G.arg

Returns:

    A table indexed by parser.arguments[name]

Usage:

    options = my_parser:get_arguments()

Tables

template_argument
Argument Template. Sample argument table for passing into add_argument. long_option and short_option flags cannot be used for positional arguments, but at least one must be given for an option argument. Option arguments cannot be required and not take arguments. Positional arguments cannot take arguments themselves. Required arguments cannot specify a default value.

Fields:

  • name Argument name used as key in returned option table.
  • long_option Specifies long option style flag
  • short_option Specifies short option style flag
  • is_positional Indicaties whether argument is positional
  • has_argument Indicates whether option takes an argument
  • is_required Indicates whether argument is mandatory or not
  • default_value Specifies a default value to be used for the argument
  • description Description to be used for display_help()
  • meta_value Argument placeholder string used in display_help()
